##vera++ rules

Rules for the vera++ C/C++ linter which conform to my coding style.

###Installation

1. Install vera++

debian-based:
```
# apt-get install vera++
```

rhel-based:
```
# yum install vera++
```

2. Install profile as default
```
$ git clone https://github.com/chuckvalenza/vera_config.git
$ cd vera_config
$ ./install
$ source ~/.bashrc
$ cd ..
$ rm -rf vera_config
```
