##vera++ rules

Rules for the vera++ C/C++ lint which conform to my coding style.

###Installation

- Install vera++
debian-based:
```
# apt-get install vera++
```

rhel-based:
```
# yum install vera++
```

- Install profile as default
```
$ git clone https://github.com/chuckvalenza/vera_config.git
$ cd vera_config
$ ./install
$ source ~/.bashrc
$ cd ..
$ rm -r vera_config
```
